# 🩺 FittyFit- https://fittyfitt.vercel.app/

### AI-Powered Healthcare Assistant for Elderly Care

FittyFit is an AI and Machine Learning-based healthcare application that helps senior citizens manage their daily health through intelligent assistance, medication tracking, food analysis, and emergency support. The system combines **Artificial Intelligence, Machine Learning, NLP, and Computer Vision** to provide personalized healthcare recommendations.

## ✨ Features

* 🤖 AI Health Assistant
* 🧠 Personalized Health Recommendations
* 🥗 AI Food Analysis (Image & Text)
* 💊 Smart Medication Tracker
* 🎤 Voice Health Summary
* 🚨 Emergency SOS & Caregiver Alerts
* 👤 Secure JWT Authentication

## 🤖 AI & ML Implementation

* **Artificial Intelligence:** Personalized health assistant and recommendation system.
* **Machine Learning:** Health pattern analysis and predictive recommendations.
* **Natural Language Processing (NLP):** Understands user queries and generates intelligent responses.
* **Computer Vision:** Analyzes food images to provide nutritional recommendations.

## 🛠 Tech Stack

| Layer          | Technology                                  |
| -------------- | ------------------------------------------- |
| Frontend       | React.js, Tailwind CSS                      |
| Backend        | FastAPI (Python)                            |
| Database       | MongoDB                                     |
| AI/ML          | Python, OpenAI GPT-4o, NLP, Computer Vision |
| Authentication | JWT                                         |

## 🚀 Run Locally

### Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn server:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm start
```
